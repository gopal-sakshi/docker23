syntax for a grok pattern ====>      %{SYNTAX:SEMANTIC}

SYNTAX      = name of the pattern
SEMANTIC    = identifier (duration23 (or) clientIp)

3.44            will be matched by the NUMBER pattern 
55.3.244.1      will be matched by the IP pattern
